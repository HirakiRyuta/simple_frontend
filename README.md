# simple-frontend

## 環境構築
```
yarn install
```

### 開発環境で起動
```
yarn start
```
アクセスは http://localhost:4000/

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```

### Run your unit tests
```
yarn run test:unit
```
